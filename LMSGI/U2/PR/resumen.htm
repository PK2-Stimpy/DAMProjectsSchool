Resumen HTML

1. Estructura básica

Para hacer un documento HTML debes saber la estructura básica.

Aquí abajo te mostramos cuál es la estructura básica de HTML.


El siguiente código va a mostrar por pantalla un texto.

TEXTO

<!DOCTYPE html>
<html>
    <head>
        <title>Prueba</title>
    </head>
    
    <body>
        <p>TEXTO</p>
    </body>
</html>

La etiqueta <p> es para definir textos.

Vamos a cambiarla a por ejemplo Patatas con ajo.

Patatas con ajo

<!DOCTYPE html>
<html>
    <head>
        <title>Prueba</title>
    </head>
    
    <body>
        <p>Patatas con ajo</p>
    </body>
</html>

2. Etiquetas de cabecera

Vamos a ver unas etiquetas que son muy útiles.


La primera de todas consta de la etiqueta <meta>.

Con esta etiqueta podemos definir cosas de la página, como por ejemplo:


* Autor, descripción y palabras clave de la página:

<head>
    <meta content="AUTOR" name="author">
    <meta content="DESCRIPCIÓN" name="description">
    <meta content="clave1, clave2, clave3, ..." name="keywords">
</head>

* También, podemos especificar la codificación de la página con la etiqueta meta:

<head>
    <meta http-equiv="content-type" content="text/html; charset=UTF-8">
</head>

** Esto nos servirá si queremos poner carácteres especiales, tildes, ...

4. Listas

Para definir una lista, se puede hacer una lista desordenada:

  • Elemento 1
  • Elemento 2
  • Elemento 3
<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>

O también, para una lista ordenada:

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
<ol>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ol>

Y por último, tenemos listas de definiciones:

Hola
Hola
Wifi
Wifi
<dl>
    <dt>Hola</dt>
    <dd>Hola</dd>

    <dt>Wifi</dt>
    <dd>Wifi</dd>
</dl>

6. Links

Podemos referenciar otras webs mediante la etiqueta <a>.

Misma pestaña
Nueva pestaña 
<a href="https://google.com" target="_self">Misma pestaña</a>
<br>
<a href="https://amazon.com" target="_blank">Nueva pestaña</a>

7. Tablas

Con <table> podemos crear tablas.

Para definir una fila usamos <tr>, y para una celda <td>.


Cada celda tiene 2 parámetros para expandirlas, colspan y rowspan.

También, podremos especificar su borde con border.

Celda 1 Celda 2
Celda 3 Celda 4 
<table border="1">
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>

    <tr>
        <td>Celda 3</td>
        <td>Celda 4</td>
    </tr>
</table>

8. Imágenes

Puedes insertar imágenes con <img>, añadiendo el parámetro src, podemos especificar la URL de la imágen.

El parámetro <alt> sirve para especificar de que va la imágen.

Y, con <width> y <height> podemos especificar el tamaño.

Imágen 100 ancho x 100 alto

<p>Imágen 100 ancho x 100 alto</p>
<img src="https://media.tenor.com/gQV5VzHLWQIAAAAd/among-us-sus.gif" 
    alt="Amogus" width="100" height="100">

9. Videos

Podemos agregar videos con <video>, en src especificamos la URL del vídeo.

En width y height el tamaño del video.

También tenemos controls, autoplay, loop y muted como parámetros.

Y, si el video es de una plataforma de streaming de video, podemos usar <iframe>.

Vídeo 200 ancho x 200 alto

<p>Vídeo 200 ancho x 200 alto</p>
<video width="200" height="200" controls 
    src="http://node2.furryporn.fun/Videos/0.%20Projectos/
    6.%20Especial%20Navidad%202021/Source%20files/Intro/
    video%20(1).mp4">Could not load video.</video>

10. Formularios

Podemos agregar formularios con <form>.

Sus parámetros son method que puede ser GET o POST,

> target que puede ser _self o _blank,

> action que es una URL y

> type que puede ser uno de los siguientes:

  • button
  • checkbox
  • color
  • date
  • datetime-local
  • email
  • file
  • hidden
  • image
  • month
  • number
  • password
  • radio
  • range
  • reset
  • search
  • submit
  • tel
  • text
  • time
  • url
  • week

Busca en Google!

<h2><b>Busca en Google!</b></h2>
<form action="https://google.com/s" 
        method="GET" target="_blank">
    <label for="q">Busca:</label>
    <input type="text" name="q" id="q" placeholder="Escribe algo!">
    <input type="submit" value="Buscar!">
</form>
